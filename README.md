# ExpenceTrackerApp

A lightweight Java-based application for tracking expenses. The project currently includes a single core Java file and an MIT license, serving as a minimal foundation for building a full expense management tool.

## Features
- Basic expense tracking logic  
- No external dependencies  
- Simple and beginner-friendly structure  

## Repository Structure
- `ExpenseTrackerApp.java` – Main program file  
- `LICENSE` – MIT License  
- `README.md` – Project documentation  

## Getting Started
1. Clone the repository  
2. Compile using `javac ExpenseTrackerApp.java`  
3. Run with `java ExpenseTrackerApp`
